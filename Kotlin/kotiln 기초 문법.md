## 1. 변수
> ### var : 변경 가능한 변수값  

> ### var : 변경 가능한 변수값
```kotlin
main(){
  var a: Int = 123 // 자료형 선언시 -> 변수: type
  print(a)
}
```
```kotlin
fun main(){
  val b: Int = 1232
  b = 3 // 중간에 값을 못바꾸기 때문에 에러발생
  print(b)
}
```
>### ? : 변수의 값이 null일 수 있다는 것을 표시(?를 표시 하지 않으면 선언시 null이 불가능)  
```kotlin
fun main(){
    val a: Int? = null
    print(a)
}
// 출력 : null
```
